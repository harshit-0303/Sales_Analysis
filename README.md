## Example Code

```python
# Create Total Amount column
df['TotalAmount'] = df['Price'] * df['Quantity']

# Group by city
revenue_by_city = df.groupby('City')['TotalAmount'].sum()

# Normalize Price column
price_mean = np.mean(df['Price'])
price_std = np.std(df['Price'])
df['Price_normalized'] = (df['Price'] - price_mean) / price_std
````
